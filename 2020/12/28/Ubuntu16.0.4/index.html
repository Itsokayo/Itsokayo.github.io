Ubuntu 16.0.4 开启 ssh

发表于 2020-12-28 阅读次数：

默认情况下Ubuntu 的 ssh 服务是关闭状态的，所以需要手动打开

首先检查服务是否安装
lsof -i:22

如果有输出就是端口开放了，如果没有输出就要需要安装服务

安装服务
apt-get install openssh-server openssh-client
修改sshd配置文件
sudo vi /etc/ssh/sshd_config

将配置修改如下

PermitRootLogin yes
重启服务
service ssh restart
给root用户配置密码
passwd

之后尝试可以root登陆，GO！
